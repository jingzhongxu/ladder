Substrate Node for Ladder network

## Hacking

1. environment

Refer to [Substrate](https://github.com/paritytech/substrate#building)

2. build

Then build the code:

```
./build.sh

cargo build
```

You can start a development chain with:
```
cargo run -- --dev
```
